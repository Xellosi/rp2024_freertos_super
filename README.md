## 中文版

`rp2024_freertos_super` 是一個面向 Raspberry Pi Pico / RP2040 的 Super Project，用來集中管理韌體開發所需的核心依賴與實驗工作區。  
專案以 Git submodule 收納 `pico-sdk`、`FreeRTOS-Kernel` 與 `rp-hal`，並在 `workplace/` 內維護實際應用與範例。

<a id="en"></a>
## English Version

`rp2024_freertos_super` is a Super Project for Raspberry Pi Pico / RP2040 firmware development, designed to centralize core dependencies and application workspaces.  
It manages `pico-sdk`, `FreeRTOS-Kernel`, and `rp-hal` via Git submodules, while keeping real applications and experiments under `workplace/`.
